## 🚀 如何运行

### 1. 安装依赖

```bash
cd ai-pic-frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问应用

打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 🔧 技术栈

- **框架**: Next.js 16.1.3 (App Router)
- **UI库**: React 19.2.3
- **样式**: TailwindCSS 4.0
- **语言**: TypeScript
- **开发工具**: ESLint, Turbopack

## 📱 页面功能详情

### 首页 (`/`)

- 系统介绍和功能展示
- 快速导航到主要功能
- 响应式设计

### 登录页面 (`/login`)

- 邮箱密码登录
- 表单验证
- 记住登录状态
- 跳转到注册页面

### 注册页面 (`/register`)

- 用户名、邮箱、密码注册
- 完整的表单验证
- 密码确认功能
- 跳转到登录页面

### 任务管理页面 (`/tasks`)

- 创建新的图片生成任务
- 任务列表展示
- 实时状态追踪
- 支持多AI平台选择
- 任务详情查看

### 图片画廊页面 (`/gallery`)

- 图片网格展示
- 搜索和筛选功能
- 图片详情模态框
- 下载功能
- 标签系统
