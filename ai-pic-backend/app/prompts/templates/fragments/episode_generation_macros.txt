{#- ============= FORMAT-SPECIFIC REQUIREMENTS ============= -#}
{%- macro format_requirements(format_type="default") -%}
{% if format_type == "short_drama" -%}
## 短剧格式要求
1. **节奏紧凑**：每分钟至少一个情节点或情绪转折
2. **爽点密集**：每集必须有明确的"爽点落地"时刻
3. **卡点结尾**：结尾必须留下悬念，驱动观众追更
4. **可拍摄性**：所有场景必须可实际拍摄，避免抽象描述
5. **投流友好**：关键片段适合剪辑为15/30/60秒素材
{% elif format_type == "film" -%}
## 电影格式要求
1. **完整叙事**：在有限时长内完成完整故事弧
2. **情绪递进**：三幕结构清晰，情绪层层递进
3. **视觉冲击**：注重关键场景的视觉表现力
4. **主题深度**：探索人物内心与主题意义
{% elif format_type == "tv_series" -%}
## 电视剧格式要求
1. **人物塑造**：深入刻画角色成长与变化
2. **多线并行**：主线与支线交织，丰富叙事层次
3. **节奏把控**：张弛有度，适合连续观看
4. **伏笔回收**：合理设置伏笔并在后续回收
{% endif -%}
{%- endmacro -%}

{#- ============= SCENE STRUCTURE ============= -#}
{%- macro scene_structure_requirements() -%}
## 场景结构要求
每个场景应包含：
1. **场景标题**：简洁明确的场景描述
2. **时间地点**：明确的时空定位
3. **人物**：出场角色列表
4. **动作**：角色行为和画面描述
5. **对白**：角色台词（如有）
6. **情绪**：场景情绪基调
{%- endmacro -%}

{#- ============= DIALOGUE REQUIREMENTS ============= -#}
{%- macro dialogue_requirements() -%}
## 对白要求
1. **口语化**：台词要符合角色身份，自然口语化
2. **简洁有力**：每句台词不超过15字为佳
3. **信息量**：对白要推动剧情或揭示角色
4. **可配音**：适合TTS语音合成
5. **去书面语**：避免过于文学化的表达
{%- endmacro -%}
