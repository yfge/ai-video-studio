你是一个专业的剧本审核专家。请审核以下对白和舞台指示的分类，找出被错误分类的内容并修正。

关键判断标准：
1. **对白 (dialogues)**: 角色说的话，直接引语，有明确的说话人。示例：
   - "嗯？电流读数异常波动。" ← 这是角色的内心独白或台词
   - "搞什么……电磁干扰？" ← 这是角色的疑问/反应
   - 任何带有疑问、感叹、对话特征的句子

2. **舞台指示 (stage_directions)**: 描述动作、场景变化、镜头、音效等非对白内容。示例：
   - "压力测试进行中。屏幕上的数据流突然加速。" ← 场景描述
   - "突然，一道蓝色电流脉冲划过房间。" ← 动作描述
   - "镜头切至角落的旧物堆。" ← 镜头指示

常见错误模式：
- 角色的内心独白被放入舞台指示
- 角色的反应性语句（如"嗯？""什么！""怎么回事"）被当作描述
- 带有省略号、感叹号、问号的直接言语被误分类

请检查以下内容：

当前对白列表：
{{ dialogues | tojson }}

当前舞台指示列表：
{{ stage_directions | tojson }}

角色列表：
{{ characters | tojson }}

请返回修正后的 JSON，格式如下：
{
  "dialogues": [...],  // 修正后的对白列表，每个包含 scene_number, character, content, emotion(可选)
  "stage_directions": [...],  // 修正后的舞台指示列表
  "corrections": [  // 记录所有修正
    {
      "original_type": "stage_direction",  // 原来的类型
      "corrected_type": "dialogue",  // 修正后的类型
      "content": "...",  // 被修正的内容
      "reason": "..."  // 修正原因
    }
  ]
}

注意：

- 确保每个场景至少有对应的对白
- 保持原有的 scene_number 不变
